Predict Iris Species with Machine Learning

Enter the flower measurements and let our machine learning model predict the Iris species in seconds.













About This Project:

The Iris Flower Classifier is a demonstration of a machine learning model deployed as a web application. The model predicts the species of an Iris flower based on four botanical measurements:

✓ Sepal Length
✓ Sepal Width
✓ Petal Length
✓ Petal Width

The prediction is made using a KNN classification algorithm written with JavaScript.

How It Works:

✓ Enter the four flower measurements.
✓ Click Predict.
✓ The algorithm compares the input with the Iris dataset.
✓ The predicted Iris species is displayed.

Built with HTML, CSS, JavaScript and a custom KNN algorithm using the Iris dataset.
