Predict Iris Species with Machine Learning

Upload the flower measurements and let our machine learning model predict the Iris species in seconds.
















About This Project:

The Iris Flower Classifier is a demonstration of a machine learning model deployed as a web application. The model predicts the species of an Iris flower based on four botanical measurements:
✓ Sepal Length
✓ Sepal Width
✓ Petal Length
✓ Petal Width
The prediction is made using a trained classification model and returned instantly.

It Works:

✓ Enter the four flower measurements.
✓ Click Predict.
✓ The model analyzes the inputs.
✓ The predicted Iris species is displayed.

Built with Python, FastAPI (or Flask), and a Machine Learning model using the Iris dataset.
